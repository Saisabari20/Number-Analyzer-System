=========================================
        NUMBER ANALYZER SYSTEM
=========================================

A Python-based console application to analyze and classify a number based on 
its mathematical properties.

-----------------------------------------
PROJECT FEATURES
-----------------------------------------
✔ Check Positive / Negative / Zero 

✔ Check Even or Odd

✔ Prime Number Checker

✔ Palindrome Checker

✔ Leap Year Checker

✔ Armstrong Number Checker

✔ Pronic Number Checker

✔ Sum of Digits

✔ Reverse of Number

✔ Count of Digits

✔ Factorial Calculation

✔ Perfect Number Checker

✔ Harshad Number Checker

✔ Find All Factors of a Number

-----------------------------------------
TECHNOLOGIES USED
-----------------------------------------
Language : Python 3

Platform : Console Application

-----------------------------------------
HOW TO RUN
-----------------------------------------
1. Install Python 3 on your system.
2. Save the program file as number_analyzer.py
3. Open Command Prompt / Terminal.
4. Run the program using:

   python number_analyzer.py

6. Enter any integer when prompted.
7. The system will display the complete analysis report.

-----------------------------------------
PROJECT OBJECTIVE
-----------------------------------------
To develop a beginner-friendly Python mini project that analyzes a given number 
and displays various mathematical properties using modular programming.
